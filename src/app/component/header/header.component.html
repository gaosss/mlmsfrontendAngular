This project is a interview demo for full stack or front end/back end developer position.


Skill set involved is as below:
1. Front end: : Angular7, JavaScript, TypeScript, CSS, HTML5, SCSS, WebPack, jQuery, Redux, RESTful, JSON.
2. Back end: Restful API, Java, SpringBoot, MySQL.
3. Front end is published on AWS S3; API is running on AWS EC2 ubuntu server; Database is AWS RDS.
